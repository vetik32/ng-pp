Key Description
script type="text/javascript" language="javascript" Indicates that the ZZZ script URL contains JavaScript.
https://pixel.ZZZ.com/partner/ The address of the ZZZ tag server. ZZZ's scripts are secure, so the protocol must be https, not http.
{{getPartnerKey()}} Your ZZZ partner key is: {{getPartnerKey()}} - it must be included in the request URL as shown.
fs Indicates the type of action being recorded. See below for the available types.
key1=value1&key2=value2&... The rest of the key/value pairs are specific to the type of action being recorded. You are responsible for inserting the values based on the user's actions.
